# 🧪 Interactive Periodic Table

An interactive and modern Periodic Table web application built with HTML, CSS, and JavaScript. It provides a clean, responsive, and color-coded interface to explore chemical elements, including their symbols, names, and groups.

---

## 🚀 Features
* **Color-Coded Groups:** Elements are categorized beautifully based on their chemical series (Alkali metals, Noble gases, Transition metals, etc.).
* **Hover Zoom Effect:** Clean micro-interactions when hovering over elements.
* **Modern UI:** Built with a flat design, clean borders, and premium typography.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure for the periodic grid.
* **CSS3:** Custom styling, layout management, and smooth hover transitions.
* **JavaScript:** (Coming Soon) For dynamic popups and chemical property search.
